Hi this was my project for the 2026 Glitch x Google DeepMind hackathon @ UCLA

Most police stations don't have dedicated forensic artists and instead rely on complicated software to create composite sketches. Suspect Sketch is an attempt at making composite sketch generation more accessible. It's currently limited by an insufficient face characteristic list and Nano Bannana's capabilities, but switching to Stable Diffusion / SDXL with ControlNet could be promising.
